# .brewfile

My Homebrew settings

## Get started

```
$ git clone git@github.com:YutoMizutani/.brewfile.git
$ cd .brewfile
$ make restore
```

## Install Homebrew

`$ make install`

## Update packages

`$ make update`

## Restore packages

`$ make restore`

## Export packages

`$ make export`

## Cleanup Homebrew

`$ make cleanup`

## Cleanup unused packages

`$ make cleanup-packages`
